
Visit Livno Web Project
------------------------

This folder contains the base structure for the Visit Livno website including:
- HTML & CSS files
- Images and tourist maps
- Embedded promotional video
- Printable PDF brochure

How to use:
1. Upload to GitHub Pages (or any hosting platform).
2. Link to custom domain if needed (see CNAME file).
3. Edit index.html and style.css to update content.

Enjoy!
